# Toward a Unified Study of Humanity

Semut begins with a question.

> What is a human?

This question is not new.

Biology, psychology, anthropology, sociology, economics, political science, philosophy, theology, artificial intelligence, and many other fields all attempt to answer it.

Each discipline studies humans from a different perspective.

Biology studies living organisms.

Psychology studies the mind.

Economics studies decisions.

Sociology studies groups.

Political science studies governance.

Artificial intelligence studies which human capabilities can be delegated.

Philosophy asks what all of this means.

These perspectives are valuable.

Semut does not attempt to replace them.

Instead, Semut asks a different question.

Can these perspectives be coordinated within a common framework?

One possible hypothesis is that many human systems emerge from repeated actions.

Actions leave traces.

Traces become memory.

Memory accumulates into knowledge.

Knowledge shapes norms.

Norms become institutions.

Institutions influence civilizations.

Different disciplines observe different parts of this process.

Semut explores whether this progression provides a useful coordination framework rather than a replacement for existing theories.

If successful, Semut is not another discipline.

It is a coordination framework connecting many disciplines around the same enduring question:

> What is a human?

Whether this hypothesis is correct remains an open question.

Semut exists to explore it.
